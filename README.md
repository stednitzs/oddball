# oddball
Notebooks used for analysis and to generate figures from Wilde et al 2024, "Evidence for auditory stimulus-specific adaptation but not deviance detection in larval zebrafish brains"
Written in Julia 1.8.5 by Sarah J Stednitz

Calcium imaging data as analyzed in these notebooks is available from: https://figshare.unimelb.edu.au/articles/dataset/Data_from_Wilde_et_al_2024/27617013
